# CS Learning with Rust

Rustを使ってコンピュータサイエンスを体系的に学ぶプロジェクト

## 📚 このプロジェクトについて

このリポジトリは、データ構造とアルゴリズムを中心としたコンピュータサイエンスの基礎から応用までを、Rustで実装しながら学ぶための教材です。

## 🎯 特徴

- **体系的なカリキュラム**: 18週間で基礎から応用まで
- **実践重視**: すべての概念に対して実装課題あり
- **Rustで学ぶ**: メモリ安全性と所有権システムを理解しながらCSを学習
- **テスト駆動**: 各実装にテストケースを用意

## 🚀 クイックスタート

```bash
# プロジェクトのビルドと実行
cd cs_learning
cargo run

# テストの実行
cargo test

# Week 1-2 の学習を開始
cat GETTING_STARTED.md
```

## 📖 ドキュメント

- **[GETTING_STARTED.md](GETTING_STARTED.md)** - 学習の始め方
- **[CS_CURRICULUM.md](CS_CURRICULUM.md)** - 全体カリキュラム
- **[CLAUDE.md](CLAUDE.md)** - Claude Code 用のガイド

## 📅 カリキュラム概要

### レベル1: 基礎（6週間）
- Week 1-2: 基本データ構造（配列、リスト、スタック、キュー、ハッシュマップ）
- Week 3-4: 探索とソート（線形探索、二分探索、各種ソートアルゴリズム）
- Week 5-6: 木構造（二分木、BST、ヒープ）

### レベル2: 応用データ構造（4週間）
- Week 7-8: 高度な木構造（AVL木、赤黒木、B木、トライ）
- Week 9-10: グラフ理論（BFS、DFS、最短経路）

### レベル3: アルゴリズム設計（4週間）
- Week 11-12: 動的計画法
- Week 13-14: 貪欲法、分割統治、バックトラッキング

### レベル4: システムとメモリ（4週間）
- Week 15-16: メモリ管理
- Week 17-18: 並行処理

## 🛠️ 開発環境

### 必要なもの
- Rust 1.70 以上
- Cargo

### インストール
```bash
# Rustのインストール（未インストールの場合）
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# プロジェクトのビルド
cd cs_learning
cargo build
```

## 📝 学習の進め方

1. **カリキュラムを確認**: [CS_CURRICULUM.md](CS_CURRICULUM.md) で全体像を把握
2. **週ごとに学習**: 各週のREADME.mdで詳細を確認
3. **実装する**: `unimplemented!()` を実際のコードに置き換え
4. **テストする**: `cargo test` でテストを実行
5. **理解を深める**: 計算量を分析し、最適化を考える

## 🎓 学習目標

このカリキュラムを完了すると、以下ができるようになります:

- 主要なデータ構造とアルゴリズムを実装できる
- 問題に対して適切なデータ構造を選択できる
- 計算量（Big O記法）を理解し、評価できる
- Rustの所有権システムとメモリ安全性を理解できる
- 並行処理を安全に実装できる
